Compatibilidad con clientes de correo
HTMLOutlook (web)
outlook.com
Usa Copiar HTML y pégalo en:
  1. Configuración → Correo → Redactar y responder → Firma de correo
  2. Pega el contenido en el editor de firma
HTMLGmail
Usa Copiar HTML y pégalo en:
  1. Configuración → Ver todos los ajustes → Firma
  2. Crea o edita una firma → pega el contenido
HTMLOutlook clásico
(2016–2021)
Usa Descargar HTML, luego:
  1. Archivo → Opciones → Correo → Firmas
  2. Crea una firma y pega el HTML desde el archivo descargado
SVGApple Mail
Usa Descargar SVG (Otros formatos), luego:
  1. Mail → Ajustes → Firmas → arrastra el archivo SVG a la firma
SVGThunderbird
Usa Descargar SVG (Otros formatos), luego:
  1. Herramientas → Configuración de cuentas → Firma
  2. Activa "Adjuntar firma desde archivo" y selecciona el SVG
